# ManagementProducts — Product Management System

[![Java](https://img.shields.io/badge/Java-ED8B00?style=flat-square&logo=openjdk&logoColor=white)]()
[![MVC](https://img.shields.io/badge/Pattern-MVC-0d1117?style=flat-square)]()
[![MySQL](https://img.shields.io/badge/MySQL-4479A1?style=flat-square&logo=mysql&logoColor=white)]()

Real-time product management software connected to a relational database. Built following the MVC (Model-View-Controller) design pattern for clean code separation.

---

## Features

- **Full CRUD** — Create, Read, Update, Delete products
- **Real-time updates** — Live database synchronization
- **MVC pattern** — Clean architecture with separated concerns
- **Relational database** — Persistent storage with MySQL

## Tech stack

| Component | Technology |
|:---|:---|
| Language | Java |
| Architecture | MVC (Model-View-Controller) |
| Database | MySQL |
| Build | Maven |

## Getting started

```bash
git clone https://github.com/Luisr26/ManagementProducts.git
cd ManagementProducts

# Configure database connection in application.properties
# Run the application
./mvnw spring-boot:run
```
